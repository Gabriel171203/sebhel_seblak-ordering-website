# SEBHEL (Seblak Rachel)

Sistem pemesanan Seblak berbasis web dengan integrasi pesanan langsung ke WhatsApp.

## Fitur Utama
- Katalog menu dinamis
- Kustomisasi paket seblak dan topping
- Sistem keranjang belanja
- Checkout otomatis ke WhatsApp
- Desain responsif (Mobile First)

## Cara Menjalankan Lokal

1. Instalasi dependensi:
   ```bash
   npm install
   ```

2. Jalankan server pengembangan:
   ```bash
   npm run dev
   ```

3. Buka [http://localhost:3000](http://localhost:3000) di browser.

## Deployment
Direkomendasikan menggunakan Vercel. Pastikan telah mengatur Environment Variables yang diperlukan untuk integrasi WhatsApp dan Cloudinary (bukti bayar).
